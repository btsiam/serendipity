Serendipity

We needed a rails-like approach for the Python language. We tried Django, Flask, etc... but we needed something else... it is like Rails but simpler, faster. Built on top of Python 3 it leverages the language high performance optimizations with the famous Python syntax sugar. Performance and Code Beauty are our goals, our poetry.

It's MVC and we focus on conventions like rails, but hey, you can customize it too...

/projectX
    /c            -    Controllers
        kit.py    -    The chain-reaction kit file. Where the magic begins.
    /m            -    Models
        db.py     -    The Database Config File. Setup Your model and override mappings.
    /v            -    The Views Directory. All your pyhtml goes here.
    /h            -    Helpers. They're everywhere.
    /i            -    i18n. For you may speak many languages.
    /l            -    Libs. Not for the fainted of heart.
    config.py     -    Here we set the show.
    sdpy          -    ...and start all things.

Simplicity is the focus, the whole framework has about 1000 lines of code and is extremely configurable.

Features

Using Serendipity you get all of this, out of the box:

  • MVC Framework
  • Tenjin Template Engine w\ Cache (One of the fastest available for Python)
  • CherryPy WebServer with Standalone, WSGI and FastCGI modes
  • Security Framework with ACL and per-method permission
  • Login Backend
  • A very fast i18n support with native Python dicts
  • Helpers support
  • External libs support
  • Auto-Import Controllers (controllers inherit automatically everything from kit, as it is already loaded in memory, we simply map all dependencies to all your controllers, no more 15+ lines of importing on top of your controllers.... just do your thing and import your stuff on kit)
  • sdpy controller / model / view generator script
  • sdpy server controller (start, stop, restart the webserver)
  • sdpy CLI - A client to connect on your database and framework over the terminal
  • SQLAlchemy + SQLasagna based Model (A respectable ORM with a powerfull auto-mapper presenting automatic backref mapping)
  • Native model support for MongoDB
  • General configuration file with Native Python Objects

Portability & Customization

The framework is self-contained, which means you don't need to install it, it always stays inside your project, versioned and tailored to each of your projects specific needs.

History

The Serendipity Framework was built by ezVoice Telecom as a base for all new products including IP PBX, Distributed Call Recorders, Unified Communications Solutions, Mobile Apps and etc... The development began in March '11 and it is currently being used to drive High Availability Telecommunications systems running on several Banks, Financial Institutions, Carriers and Oil Companies both in Brazil and USA.
